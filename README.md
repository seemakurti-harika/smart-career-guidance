# Smart Career Guidance Web Application

A full-stack web application built using **Python Flask** that provides **skill-based career recommendations**, performs **skill gap analysis**, and generates **personalized learning roadmaps**. The application includes **secure authentication**, **session-based access control**, and **SQLite database integration**.
